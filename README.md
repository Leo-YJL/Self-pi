# Self-pi

Personal Pi package monorepo (`pipackage`).

## Packages

- `package/pi-coding-workflow` — Generic Pi coding workflow package with `workflow_next`, `workflow_run`, `/workflow-init`, and `/workflow-init-spec`.

## Local development

```bash
cd package/pi-coding-workflow
npm test
```

Or from repository root:

```bash
npm test
```

## GameBase local package path

GameBase loads the workflow package from:

```text
D:/YJL_AI/PI_Package/pipackage/package/pi-coding-workflow
```
